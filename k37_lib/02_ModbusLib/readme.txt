2、Modbus通用标准协议，在数采仪【串口参数设置】-【ModBus参数设置】
   ModBus参数设置说明：

			1.设备地址，默认值 1
			2.数据排序，默认值 0(int):	0 -- int  整形
										1 -- unsigned 无符号整形
										2 -- long	长整型，1032
										3 -- long_inverse	长整型，3210
										4 -- long_0123		长整型，0123
										5 -- float			浮点数，1032
										6 -- float_inverse	浮点数，3210
										7 -- float_0123		浮点数，0123
			3.开始寄存器，默认值 0
			4.寄存器数量，默认值 1
			5.485映射，	  默认值 0	：是否映射到485, 0--不映射 ，1--映射;
			6.模式，	  默认值 0	：modbus使用哪种模式，0--RTU ，1--ASCII
	仪表型号：
			功能码为 03时，选择仪表型号 4444
			功能码为 04时，选择仪表型号	3333
			
例：仪表使用modbus-RTU模式，从机地址5，数据类型 float1032，开始寄存器 40005，读2个寄存器，不映射485.

	在数采仪上设置如下：仪表型号：4444，设备地址：5，数据排序：5，开始寄存器：4，寄存器数量：2,485映射：0，模式：0;
